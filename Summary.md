# Итоги автоматизации

## Что было запланировано и что реализовано

### Запланировано:
-  Настройка окружения для автоматизации
-  Создание UI-тестов для формы оплаты
-  Интеграция с СУБД для проверки данных
-  Написание отчетной документации
-  Создание репозитория на GitHub

### Реализовано:
-  Настроено окружение: Java 17, Selenide, JUnit, Gradle
-  Создано 5 UI-тестов для различных сценариев
-  Реализованы демо-запросы к СУБД с SQL запросами
-  Создана полная отчетная документация
-  Проект размещен на GitHub

### Не реализовано:
-  Тестирование основного приложения aqa-shop.jar
-  Реальное подключение к СУБД MySQL/PostgreSQL

## Причины, по которым что-то не было реализовано

### Основная причина:
Приложение aqa-shop.jar не запускается из-за критических ошибок конфигурации Spring Boot. Многочисленные попытки настройки не увенчались успехом.

### Технические проблемы:
- Ошибка: `Could not resolve placeholder 'spring.credit-gate.url'`
- Конфигурационные файлы игнорируются приложением
- Отсутствует документация по корректной настройке
- Проблемы с загрузкой properties-файлов

## Сработавшие риски
