# План автоматизации тестирования

## 1. Автоматизируемые сценарии
- Успешная покупка тура с одобренной картой
- Отказ в покупке с отклоненной картой
- Покупка с неверными данными карты

## 2. Инструменты
- Java, Selenium, JUnit, Gradle, Allure

## 3. Риски
- Сложности в настройке окружения
- Изменения в веб-интерфейсе

## 4. Оценка времени: 5 рабочих дней
