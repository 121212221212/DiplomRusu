# Дипломный проект: Автоматизация тестирования платежной системы

## Описание проекта
Дипломный проект по автоматизации тестирования для профессии "Инженер по тестированию" в Нетологии.

Проект включает автоматизацию тестирования веб-сервиса покупки тура с использованием дебетовой карты.

## Технологии и инструменты
- **Java 17** - язык программирования
- **Selenide** - фреймворк для автоматизации UI-тестов
- **JUnit 5** - фреймворк для модульного тестирования
- **Gradle** - система сборки
- **Git** - система контроля версий
- **Chrome Driver** - для взаимодействия с браузером Chrome

## Установка и запуск

### Предварительные требования
- Java JDK 17 или выше
- IntelliJ IDEA
- Браузер Chrome
- Git

### Запуск автотестов
1. Клонируйте репозиторий:
```bash
git clone https://github.com/121212221212/DiplomRusu.git 
```

Откройте проект в IntelliJ IDEA

Запустите тесты в файле src/test/java/ru/netology/test/DebitCardTest.java
